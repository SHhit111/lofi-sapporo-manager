# Lofi Sapporo Beats — チャンネル管理ツール

## 使い方

### アプリを開く
`index.html` をブラウザにドラッグ＆ドロップ（またはダブルクリック）するだけ。

### データについて
- データはブラウザのlocalStorageに自動保存される
- 同じブラウザで開けばデータは消えない
- **別のブラウザやPCでは共有されない**（そのPCのブラウザに保存されるため）

---

## Claude Codeで改善する方法

### 1. Claude Codeをインストール（まだの場合）

**Mac:**
```bash
brew install anthropic/tap/claude-code
```

**Windows（WSL経由）:**
```bash
npm install -g @anthropic-ai/claude-code
```

### 2. このフォルダで起動

```bash
cd lofi-sapporo-manager
claude
```

### 3. 日本語で指示するだけ！

```
例：
「動画をCSVでエクスポートできるようにして」
「アナリティクスをグラフで表示して」
「SUNOプロンプトにお気に入り機能を追加して」
「投稿カレンダーに締め切りアラートを追加して」
```

Claude Codeが `index.html` を直接編集してくれます。

## 機能一覧

| タブ | 機能 |
|------|------|
| ダッシュボード | 統計サマリー・タスク管理・直近予定 |
| 動画管理 | タイトル・説明文・タグ・視聴回数の記録 |
| 投稿カレンダー | 月次カレンダーで投稿スケジュール管理 |
| SUNOプロンプト | 音楽生成プロンプトの保存・評価・コピー |
| 再生リスト | テーマ別の動画グループ管理 |
| アナリティクス | 登録者数・視聴回数の手動記録・履歴 |
