# Lofi Sapporo Beats — チャンネル管理ツール

## プロジェクト概要
YouTubeチャンネル「Lofi Sapporo Beats」(@LofiSapporoBeats) の運営を効率化するWebアプリ。
ローファイBGMチャンネル。音源はSUNO AIで生成。現在登録者2人・動画9本。

## ファイル構成
```
lofi-sapporo-manager/
├── CLAUDE.md          ← このファイル（プロジェクト説明）
├── index.html         ← メインアプリ（全機能をこの1ファイルに集約）
└── README.md          ← 使い方ガイド
```

## アプリの機能（index.html）
- **ダッシュボード** — 統計・今週タスク・直近投稿予定・視聴回数TOP3
- **動画管理** — 9本の動画のタイトル・説明文・タグ・視聴回数を管理
- **投稿カレンダー** — 月次カレンダーで投稿スケジュールを管理
- **SUNOプロンプト管理** — 音楽生成プロンプトを保存・評価・コピー
- **再生リスト整理** — テーマ別に動画をグループ化
- **アナリティクスメモ** — 登録者数・視聴回数を手動記録

## 技術仕様
- **純粋なHTML/CSS/JS** — フレームワークなし、依存ライブラリなし
- **データ保存** — localStorage（ブラウザ内に永続化）
- **フォント** — Zen Kaku Gothic New (JP) + Space Mono (英数)
- **カラーテーマ** — ダークテーマ固定
- **ビルド不要** — index.htmlをブラウザで直接開くだけで動作

## 開発上の注意点
- 1ファイル完結の設計を維持すること（index.html以外にJSやCSSファイルを作らない）
- データはlocalStorageキー `lsb_*` で管理している
- seedIfEmpty() 関数でデフォルトデータを初期化（既存データがある場合は上書きしない）
- newId() でユニークIDを生成（lsb_nextId キーを使用）
